MoneyFlix

Esse projeto me fez criar um controle de financas,responsiza de acordo com o design que recebi. Usei HTML5, CSS Grid e JavaScript para as áreas interativas.

Linguagens Utilizadas

HTML / CSS / JavaScript / React.js / PostgresSQL

VISITAR

Projeto

É uma plataforma para fazer controle de financas, tanto empresarial, quanto pessoal, voce consegue colocar novas faturas em clientes quanto saber se eles efetuaram o pagamento dentro do prazo. Tambem consegue ter o controle quais os clientes que estao pagando em dia, e quais estao ainda sem pagar.
